# DrumKit.js

## Description

GIVEN a webpage consisting of simple buttons

WHEN I press certain buttons
THEN a corresponding sound emits

WHEN I press 'a', 's', 'd', 'f', 'g', 'h', 'j', 'k', or 'l'
THEN I hear a corresponding musical sound
